# MindPaw 帮助中心

欢迎使用 MindPaw！我们提供两种方式来帮助你提升专注力、调节情绪、改善睡眠。

---

## 一、轻训练工具（快速上手）

这些工具每天只需几分钟，帮你建立习惯：

- **[微番茄钟](/tools/tomato)**  
  5分钟专注计时器，完成后自动记录。每天3次，解锁徽章。

- **[每日专注力报告](/tools/report)**  
  记录专注次数和干扰因素，生成进步报告，可分享图片。

- **[情绪急救卡片](/tools/emotion)**  
  5个立刻能用的情绪调节方法，可保存为图片随身带。

- **[考前减压助手](/tools/exam)**  
  减压建议 + 考友互助留言板，分享可解锁更多次数。

- **[专注力打卡日历](/tools/calendar)**  
  日历打卡 + 连续天数统计，培养长期习惯。

**使用建议**：先从微番茄钟开始，每天完成1-3个，再配合打卡日历，效果更好。

## 二、深度心理助手（扣子版）

如果你需要更深入的情绪梳理，可以试试我们的 **深度心理助手**（基于扣子平台）。

- **体验地址**：https://www.coze.cn/s/SpDail_Ok3Q/
- **使用技巧**：直接说你的困扰，比如“我最近很焦虑”“失眠怎么办”“该考试了学不进去”等等任何情绪问题，它会引导你一步步梳理并给出方法。

## 三、常见问题

**Q：这些工具收费吗？**  
A：完全免费，没有隐藏费用。我们通过赞助维持开发。

**Q：数据会保存多久？**  
A：所有数据保存在你自己的浏览器中（localStorage），不会上传到服务器，清除浏览器缓存会丢失。

**Q：工具用起来卡怎么办？**  
A：检查网络，或尝试刷新页面。如果频繁出问题，请通过反馈渠道告诉我们。

**Q：我还能在哪里找到 MindPaw？**  
A：
