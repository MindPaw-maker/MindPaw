# MindPaw 帮助中心

欢迎使用 MindPaw！我们提供两种方式来帮助你提升专注力、调节情绪、改善睡眠。

---

## 一、轻训练工具（快速上手）

这些工具每天只需几分钟，帮你建立习惯。建议按顺序尝试：

### 1. 微番茄钟
- **作用**：5分钟专注，训练专注力
- **怎么用**：点击“开始专注5分钟”，倒计时结束后自动记录一次
- **建议**：每天完成3个番茄钟，坚持一周会有明显变化

👉 [体验微番茄钟](https://h7ihprbsjuux.space.minimaxi.com/)

### 2. 专注力打卡日历
- **作用**：记录每天专注情况，培养长期习惯
- **怎么用**：完成专注后，点击日历格子标记“已专注”
- **建议**：连续打卡7天会弹出庆祝动画

👉 [体验专注力打卡日历](https://xqf9uglffmy0.space.minimaxi.com/)

### 3. 每日专注力报告
- **作用**：回顾今天的专注表现，生成进步报告
- **怎么用**：输入专注次数、选择干扰因素，点击“生成报告”
- **建议**：每周看一次报告，对比自己的进步

👉 [体验每日专注力报告](https://1r9jcc61e4y5.space.minimaxi.com/)

### 4. 情绪急救卡片
- **作用**：焦虑时立刻可用的调节方法
- **怎么用**：选择一种方法，点击“保存卡片”到手机，随时查看
- **建议**：把卡片设成手机壁纸，需要时看一眼

👉 [体验情绪急救卡片](https://ipijwl28v37g.space.minimaxi.com/)

### 5. 考前减压助手
- **作用**：减压建议 + 考友互助留言板
- **怎么用**：点击“开始减压”获取建议，用完3次可分享解锁更多
- **建议**：在留言板写下你的减压妙招，帮助其他同学

👉 [体验考前减压助手](https://cqphfrnoyslk.space.minimaxi.com/)

## 三、常见问题

**Q：这些工具收费吗？**  
A：完全免费，没有隐藏费用。我们通过赞助维持开发（底部有“赞助我们”按钮）。

**Q：数据会保存多久？**  
A：所有数据保存在你自己的浏览器中，清除缓存会丢失。建议定期截图保存报告。

**Q：工具用起来卡怎么办？**  
A：检查网络，或刷新页面。如果频繁出问题，请通过底部反馈渠道告诉我们。

**Q：我还能在哪里找到 MindPaw？**  
A：
