## 🤖 AI自动化运营系统（开源模板）

我们基于扣子(Coze)平台搭建了一套**全自动行业情报收集与内容生成系统**，包含五个智能体 + 工作流：

| 智能体 | 功能 | 输出示例 |
|--------|------|----------|
| AI早报助手 | 每日抓取AI心理/专注力热点 | 飞书群每日8:00推送 |
| 竞品周报助手 | 追踪竞品动态（心大陆、Endel等） | 每周一推送 |
| 用户洞察助手 | 抓取知乎/小红书用户痛点 | 每周二推送 |
| 行业趋势助手 | 政策、技术、学术、市场动态 | 每周三推送 |
| 产品优化助手 | 综合前四份报告生成P0/P1建议 | 每周四推送 |

> 💡 **这套系统已完全跑通，你可以直接复用到自己的项目中。**  
> 如需获取工作流配置和详细指导，请在 [Issues](https://github.com/MindPaw-maker/MindPaw/issues) 中提出，我会提供完整的导入步骤和配置说明（不包含可导入文件，需自行在扣子平台操作）。
