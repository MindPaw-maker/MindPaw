# MindPaw 心理专家

这里是我们深度优化的AI心理助手，包含20个细分场景和多轮深度对话功能。

## 如何使用

点击下方链接，即可直接与专家对话（需登录MiniMax账号）：

👉 [**心理助手·焦虑多轮对话**](https://agent.minimaxi.com/experts?preview_expert_id=377569089286211)

## 专家功能

- 覆盖20+心理场景（考试焦虑、职场压力、人际关系、失恋、家庭困扰等）
- 5阶段深度提问（触发事件→持续时间→身体反应→影响程度→核心信念）
- 基于CBT/ACT/传统文化的个性化建议
- 危机干预与转介流程

## 使用提示

- 专家完全免费，欢迎使用
- 如果遇到危机情况，AI会提供心理援助热线，并建议寻求专业帮助
- 请注意：AI可能会根据对话内容进行心理风险评估，但不会随意询问极端问题。如果你感到不适，可以随时结束对话
- 你的对话数据仅用于本次会话，不会被记录
